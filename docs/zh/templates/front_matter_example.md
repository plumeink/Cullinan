# Front-matter 示例（模板）

将以下块放在每个公开文档的顶部（替换值）：

```
title: "<简短标题>"
slug: "<路径友好 slug>"
module: ["cullinan.core"]
tags: ["getting-started", "ioc"]
author: "姓名 <github>"
reviewers: ["姓名 <github>"]
status: draft
locale: zh
translation_pair: "docs/<file>.md"
related_tests: ["tests/test_real_app_startup.py"]
related_examples: ["examples/hello_http.py"]
estimate_pd: 2.0
last_updated: "2025-11-18T00:00:00Z"
pr_links: []
```

说明：保持该块简洁，便于机器解析。
